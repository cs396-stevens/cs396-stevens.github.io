RSA Signatures

This challenge is an RSA forgery problem presented in a worker/dispatcher format.

Here, the /challenge/dispatcher script is your â€œoracleâ€ that will sign any message (given as a command line argument) except the command â€œflagâ€.

It takes its input as a base64-encoded string, which it will decode into the actual message to be signed. It will then sign the message using a pre-generated private key, encode the signature in base64, and print the result.

The dispatcher script is as follows:

#!/usr/bin/exec-suid -- /usr/bin/python3 -I

import sys

from base64 import b64encode, b64decode

n = int(open("/challenge/key-n").read(), 16)
d = int(open("/challenge/key-d").read(), 16)

if len(sys.argv) < 2:
    print(f"Usage: {sys.argv[0]} [command-b64]")
    sys.exit(1)

command = b64decode(sys.argv[1].strip("\0"))

if b"flag" in command:
    print(f"Command contains 'flag'")
    sys.exit(1)

signature = pow(int.from_bytes(command, "little"), d, n).to_bytes(256, "little")
print(f"Signed command (b64): {b64encode(signature).decode()}")

The base64-encoded, signed command can then be passed as a command line argument to the /challenge/worker script, which decode the signature, then decrypt it using the public key to obtain the original message. Thus, the worker should only accept commands that have been signed by the dispatcher.

If this message was â€œflagâ€, it will print the flag, otherwise it will do nothing.

Note that the original message is not passed to the worker in addition to the signature.

The worker script is as follows:

#!/usr/bin/exec-suid -- /usr/bin/python3 -I

import sys

from base64 import b64decode

n = int(open("/challenge/key-n").read(), 16)
e = int(open("/challenge/key-e").read(), 16)

if len(sys.argv) < 2:
    print(f"Usage: {sys.argv[0]} [signature-b64]")
    sys.exit(1)

signature = b64decode(sys.argv[1])
c = int.from_bytes(signature, "little")
assert c < n, "Message too big!"
command = pow(c, e, n).to_bytes(256, "little").rstrip(b"\x00")

print(f"Received signed command: {command}")
if command == b"flag":
    print(open("/flag").read())

SHA 1

This challenge requires you to find a partial hash collision with the flag under SHA256.

The run script is as follows:

#!/usr/bin/exec-suid -- /usr/bin/python3 -I

import hashlib

flag = open("/flag").read()
prefix_length = 6
flag_hash = hashlib.sha256(flag.encode("latin")).hexdigest()
print(f"{flag_hash[:prefix_length]=}")

collision = bytes.fromhex(input("Colliding input? ").strip())
collision_hash = hashlib.sha256(collision).hexdigest()
print(f"{collision_hash[:prefix_length]=}")
if collision_hash[:prefix_length] == flag_hash[:prefix_length]:
    print("Collided!")
    print(flag)
